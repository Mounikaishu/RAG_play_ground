DSA INTERVIEW PREPARATION — TOP 50 QUESTIONS BY CATEGORY

ARRAYS (10 Questions)
1. Two Sum — Use HashMap for O(n) solution
2. Best Time to Buy and Sell Stock — Single pass, track minimum price
3. Maximum Subarray (Kadane's Algorithm) — O(n) dynamic programming
4. Product of Array Except Self — Prefix and suffix products, O(n) no division
5. Container With Most Water — Two pointer approach
6. 3Sum — Sort + two pointers, handle duplicates carefully
7. Merge Intervals — Sort by start time, merge overlapping
8. Next Permutation — Find rightmost ascending pair
9. Rotate Array — Reverse approach for O(n) time, O(1) space
10. Trapping Rain Water — Two pointer or stack approach

STRINGS (8 Questions)
11. Longest Substring Without Repeating Characters — Sliding window + HashSet
12. Valid Parentheses — Stack-based matching
13. Group Anagrams — Sort each string as key, or use character count
14. Longest Palindromic Substring — Expand around center
15. Minimum Window Substring — Sliding window with frequency map
16. String to Integer (atoi) — Handle edge cases: whitespace, sign, overflow
17. Longest Common Prefix — Vertical scanning approach
18. Palindrome Partitioning — Backtracking with DP for palindrome check

LINKED LISTS (6 Questions)
19. Reverse Linked List — Iterative (3 pointers) or recursive
20. Merge Two Sorted Lists — Compare and link nodes
21. Detect Cycle in Linked List — Floyd's tortoise and hare
22. Remove Nth Node From End — Two pointers with gap of n
23. Intersection of Two Linked Lists — Length difference or two-pointer
24. Reorder List — Find middle, reverse second half, merge

TREES (8 Questions)
25. Maximum Depth of Binary Tree — DFS recursive
26. Level Order Traversal — BFS with queue
27. Validate Binary Search Tree — Inorder traversal or range checking
28. Lowest Common Ancestor — Recursive divide and conquer
29. Binary Tree Right Side View — BFS, last node per level
30. Serialize and Deserialize Binary Tree — BFS or preorder DFS
31. Diameter of Binary Tree — Track max at each node during DFS
32. Construct Binary Tree from Preorder and Inorder — Recursive partition

GRAPHS (8 Questions)
33. Number of Islands — DFS/BFS on grid
34. Course Schedule — Topological sort with cycle detection
35. Clone Graph — BFS/DFS with visited HashMap
36. Word Ladder — BFS for shortest transformation
37. Pacific Atlantic Water Flow — Multi-source BFS from borders
38. Shortest Path in Binary Matrix — BFS
39. Network Delay Time — Dijkstra's algorithm
40. Accounts Merge — Union-Find or DFS

DYNAMIC PROGRAMMING (10 Questions)
41. Climbing Stairs — Fibonacci variant
42. House Robber — Include/exclude DP
43. Longest Increasing Subsequence — O(n log n) with binary search
44. Coin Change — Bottom-up DP, min coins for amount
45. Edit Distance — 2D DP table
46. Decode Ways — 1D DP with constraints
47. Unique Paths — 2D DP grid
48. Word Break — DP with word set lookup
49. Partition Equal Subset Sum — 0/1 knapsack variant
50. Longest Common Subsequence — Classic 2D DP

PREPARATION TIPS
- Solve each problem independently before looking at solutions
- Understand the pattern, not just the solution
- Practice writing clean code on first attempt
- Time yourself: aim for medium problems in 20-25 minutes
- Revise solved problems every 2 weeks (spaced repetition)
- Focus on patterns: sliding window, two pointers, BFS/DFS, DP states
